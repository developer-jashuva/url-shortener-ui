# Secure URL Shortener UI

Angular frontend for the Secure URL Shortener system.  
Provides a modern interface to shorten URLs, view analytics, and manage links.

---

## Features

- URL shortening form
- Copy short URL
- Error handling UI
- URL history dashboard
- Click analytics table
- Delete links
- Responsive Bootstrap UI

---

## Tech Stack

- Angular
- TypeScript
- Bootstrap
- REST API integration

---

## How to Run

```bash
git clone https://github.com/developer-jashuva/url-shortener-ui
cd url-shortener-ui
npm install
ng serve
```

Backend must be running at:

```
http://localhost:5158
```
